## Remarque (Mermaid)
GitHub rend les diagrammes Mermaid directement dans les fichiers Markdown (blocs `mermaid`).
Pour exporter en images (slides), les mêmes blocs peuvent être collés dans un éditeur Mermaid (ex. mermaid.live) puis exportés en SVG/PNG.
